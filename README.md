# Zylos Pro

A premium AI powered business growth platform.

## About
This is a showcase of Zylos Pro, an AI driven platform built to help small businesses, startups, and entrepreneurs scale faster and smarter.

The full source code is private. This repository highlights the concept, features, and technology behind the platform.

## Features
- AI powered business growth tools
- Automated workflows for operations, marketing, and sales
- Real time analytics and insights dashboard
- Custom campaign builder
- Responsive, modern UI
- Performance focused architecture

## Tech Stack
- Frontend: React, Next.js, Tailwind CSS
- Animations: Framer Motion
- Backend & Integrations: Node.js, Firebase, Stripe, OpenAI APIs
- Build & Deployment: Vite, npm, GitHub Pages
